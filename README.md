**nEMU** [**n**curses interface for Q**EMU**]
===========

## Features
 * Install VM
 * Delete VM
 * Clone VM
 * Show VM status
 * Start/stop/shutdown/reset VM
 * Connect to VM via vnc
 * Show/Edit VM settings
 * USB support
 * Network via tap interfaces
