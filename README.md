Qemu Manage System
===========

## Features
 * Add guest
 * Delete guest
 * CLone guest
 * Show guest status
 * Start/stop guest
 * Connect to guest via vnc
 * Show/Edit guest hardware parametrs
 * USB support
 * Show OpenvSwitch map (TODO)
