nemu [<b>n</b>curses interface for Q<b>EMU</b>]
===========

## Features
 * Install guest
 * Delete guest
 * Clone guest
 * Show guest status
 * Start/stop guest
 * Connect to guest via vnc
 * Show/Edit guest hardware parametrs
 * USB support
 * Network via tap interfaces
