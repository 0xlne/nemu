**nEMU** [**n**curses interface for Q**EMU**]
===========

## Features
 * Install guest
 * Delete guest
 * Clone guest
 * Show guest status
 * Start/stop guest
 * Connect to guest via vnc
 * Show/Edit guest hardware parametrs
 * USB support
 * Network via tap interfaces
